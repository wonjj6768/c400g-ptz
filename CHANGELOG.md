# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-01-14

### Added
- Initial release
- Basic PTZ control for ipTIME C400G camera via ONVIF
- `step()`, `left()`, `right()`, `up()`, `down()` methods
- `start_move()` and `stop()` for continuous movement
- Configurable default step time
- Optional logging support
